# Day 45 学习总结：POST /evaluation

今天完成了 DocuAsk v2 的第八步升级：新增固定问题评估接口。

## 今天真正理解的点

评估不是临时脚本，而应该成为项目能力的一部分。

现在同一套 10 题评估可以在三个地方使用：

- Streamlit 页面展示。
- FastAPI 接口输出。
- pytest 自动验证。

这比“我手动跑过”更有说服力。

## 今天的项目变化

新增：

```text
backend/services/evaluation.py
POST /evaluation
```

接口会返回：

- 评估问题数量。
- Top-1 命中率。
- Top-k 召回率。
- 每道题的命中详情。

```text
Top-k recall: 1.0
```
