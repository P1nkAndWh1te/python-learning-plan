# Day 49 学习总结：Evaluation Retrieval Mode

今天完成了 DocuAsk v2 的第十二步升级：让 `/evaluation` 支持检索模式对比。

## 今天真正理解的点

优化检索不能只看一个问题。

如果只问：

```text
RAG 的基本流程是什么？
```

三种模式都可能表现不错。

但项目要证明“优化有效”，需要固定问题集和指标。

## 今天的项目变化

`POST /evaluation` 新增：

```text
retrieval_mode
```

现在可以比较：

```text
vector
bm25
rrf
```

## 今天的验收结果

三种模式都能跑完整 10 题评估。

未知 mode 会返回 400。
