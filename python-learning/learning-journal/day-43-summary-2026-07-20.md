# Day 43 学习总结：POST /qa

今天完成了 DocuAsk v2 的第六步升级：新增检索问答接口。

## 今天真正理解的点

`/documents` 和 `/qa` 是 RAG 后端的两个阶段：

```text
/documents：把资料写进知识库
/qa：根据问题从知识库检索资料
```

今天还没有让 LLM 生成最终答案，因为检索本身就需要单独验证。

## 今天的项目变化

后端新增：

```text
POST /qa
```

它可以根据 `collection_name` 和问题返回：

- 命中的 chunk。
- 每个 chunk 的 distance。
- 拼接后的 context。
