# Day 42 学习总结：POST /documents

今天完成了 DocuAsk v2 的第五步升级：新增文档写入接口。

## 今天真正理解的点

FastAPI endpoint 不是重新写一套逻辑，而是把已有 service 包装成可调用接口。

今天的接口链路是：

```text
HTTP 请求 -> Pydantic 请求模型 -> chunking service -> retrieval service -> Chroma
```

这说明前几天抽 service 是有意义的。

## 今天的项目变化

后端新增：

```text
POST /documents
```

这个接口可以把一段文档文本写入 Chroma，并返回：

- 文档 ID。
- collection 名。
- chunk 数量。
- 已存储 chunk 数量。

## 今天的验收结果

```text
Status code: 200
Chunk count: 6
Stored chunk count: 6
Bad request status: 400
```
