# Day 33 学习总结：Top-1 与 Top-k 指标

## 今天完成了什么

今天优化了主应用的检索评测展示。

页面现在会同时显示：

```text
Top-1 hit
Top-k recall
```

还给 BGE 模式增加了首次加载提示。

## 今天的关键理解

RAG 检索不是只看一个“命中率”。

`Top-1 hit` 关注第一名是否正确。

`Top-k recall` 关注正确资料有没有进入候选集合。

如果 top-k recall 高但 top-1 hit 低，说明资料已经找到了，但排序还可以优化。

## 当前结果

```text
Teaching keyword embedding: Top-1 60%, Top-k 100%
BGE Chinese embedding: Top-1 80%, Top-k 100%
```

## 当前判断

BGE 在排序上更好，而两种模式都能召回正确资料。

所以现在不急着继续换 embedding。下一步更应该把项目整理成一个可展示的 RAG 小作品。
