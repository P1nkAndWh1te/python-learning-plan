# Day 46 学习总结：BM25 Baseline

今天完成了 DocuAsk v2 的第九步升级：新增 BM25 关键词检索 baseline。

## 今天真正理解的点

BM25 和 embedding 检索不是一回事。

BM25 看的是词：

- 问题里的词有没有出现在文档里。
- 词在文档里出现了多少次。
- 词在所有文档里是不是稀有。
- 文档长度是否过长。

Embedding 看的是向量方向和语义相似度。
