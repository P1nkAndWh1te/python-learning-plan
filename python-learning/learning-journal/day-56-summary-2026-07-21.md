## 今天的核心认知

项目面试不是背技术名词，而是把每个技术选择讲成：

```text
为什么要做 -> 怎么做的 -> 怎么验证 -> 有什么边界
```

DocuAsk 当前最有价值的点不是“用了很多框架”，而是已经形成了可验证链路：

```text
本地文档 -> chunk -> embedding -> Chroma -> retrieval -> LLM -> sources -> evaluation
```
