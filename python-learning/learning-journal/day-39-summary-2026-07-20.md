# Day 39 学习总结：抽离 Embedding Service

## 今天完成了什么

今天把 embedding 相关逻辑从 Streamlit `app.py` 抽离到了：

```text
backend/services/embeddings.py
```

包括：

- 教学版关键词 embedding。
- BGE 中文 embedding。
- embedding mode 常量。
- collection name 映射。
- matched concepts。

## 为什么这样做

v2 的目标是让 RAG 核心逻辑逐步脱离页面。

embedding 是检索链路的核心模块。把它抽成 service 后，后续 FastAPI 后端可以复用同一套 embedding 逻辑。

## 今天的关键理解

service 层不应该依赖 Streamlit。

所以 BGE 模型缓存从：

```text
st.cache_resource
```

调整为：

```text
functools.lru_cache
```

这样它既能被 Streamlit 使用，也能被 FastAPI 使用。

## 验证结果

```text
Keyword embedding dimension: 12
BGE embedding dimension: 512
Teaching keyword embedding: 60% / 100%
BGE Chinese embedding: 80% / 100%
```

下一步建议抽离 Chroma collection 构建和 retrieval service。
