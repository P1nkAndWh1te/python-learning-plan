# Day 32 学习总结：主应用接入 BGE

## 今天完成了什么

今天把 BGE 中文 embedding 接入了 Streamlit 主应用。

主应用现在支持两种模式：

```text
Teaching keyword embedding
BGE Chinese embedding
```

评测和问答都会使用当前选择的模式。

## 为什么不是直接替换

因为教学版 embedding 仍然适合学习底层原理。

BGE 中文 embedding 更接近真实项目能力，但它有模型加载成本。

所以双模式更适合当前学习项目：

```text
教学版用于理解
BGE 用于真实效果验证
```

## 今天的关键理解

工程接入不是只把模型跑起来。

还要考虑：

- 模型加载速度。
- Streamlit 重跑机制。
- 缓存。
- 用户能否切换和对比。
- 原有功能是否被破坏。

## 当前状态

主应用已经具备真实中文 embedding 检索能力。

下一步可以继续优化：

- 页面上的评测指标。
- BGE 加载体验。
- 失败题分析。
- 是否加入 rerank。
