# Day 47 学习总结：RRF Baseline

今天完成了 DocuAsk v2 的第十步升级：新增 RRF 混合排序 baseline。

## 今天真正理解的点

RRF 不是重新计算语义相似度，也不是重新算 BM25 分数。

它融合的是“排名”：

```text
一个 chunk 在多个检索器里都排得越靠前，融合分数越高。
```

这适合把向量检索和关键词检索结合起来。

## 今天的项目变化

新增：

```text

```

当前 RRF 只作为独立 service，不影响现有 `/qa` 和 `/evaluation`。
