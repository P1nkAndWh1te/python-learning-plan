## 今日知识点

### 1. json 模块：数据交换的"通用语言"

#### 四兄弟速记

| 函数 | 参数 | 方向 | 作用 |
|------|------|------|------|
| `json.dumps(数据)` | 1 个 | 对象 → 字符串 | 字典转 JSON 字符串 |
| `json.loads(字符串)` | 1 个 | 字符串 → 对象 | JSON 字符串转字典 |
| `json.dump(数据, 文件)` | 2 个 | 对象 → 文件 | 写入 JSON 文件 |
| `json.load(文件)` | 1 个 | 文件 → 对象 | 从 JSON 文件读取 |

规律：**带 `s` 操作字符串，不带 `s` 操作文件**。

#### 完整示例

```python
import json

data = {"name": "RAG项目", "tech_stack": ["Python", "LlamaIndex"], "stars": 100}

# 写入 JSON 文件
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False)  # 保留中文

# 从 JSON 文件读取
with open("data.json", "r", encoding="utf-8") as f:
    loaded = json.load(f)
    print(loaded["name"])  # RAG项目
```

#### 关键注意事项

- `ensure_ascii=False`：不加这个的话，中文会被编码成 `\u9879\u76ee`，人读不了
- `json.load()` 返回的已经是字典，不要再 `dict()` 包一层
- `json.dump()` 需要两个参数（数据 + 文件），`json.dumps()` 只需要一个（数据）

### 2. Pandas 入门：表格数据处理

#### 六大核心操作

| 操作 | 代码 | 作用 |
|------|------|------|
| 读取文件 | `pd.read_csv("文件.csv")` | 把 CSV 装进 DataFrame |
| 预览 | `df.head()` | 看前 5 行 |
| 统计 | `df.describe()` | 最大值、最小值、平均值、标准差 |
| 选列 | `df["列名"]` | 取某一列 |
| 筛选 | `df[df["分数"] > 85]` | 只保留符合条件的行 |
| 分组 | `df.groupby("城市")["分数"].mean()` | 按某列分组后求平均值 |

#### 完整示例

```python
import pandas as pd

df = pd.read_csv("students.csv")

print(df.head())           # 前 5 行
print(df.describe())       # 统计概览

# 筛选分数大于平均值
avg = df["分数"].mean()
print(df[df["分数"] > avg])

# 按城市分组，算平均分
print(df.groupby("城市")["分数"].mean())
```

#### 筛选条件的工作原理

```
df["分数"]              →  [85, 92, 78, 88, 95]
df["分数"] > 87         →  [False, True, False, True, True]
df[布尔列表]             →  只保留 True 的行
```

## 今日踩过的坑

| 错误 | 原因 | 修复 |
|------|------|------|
| `json_loads` 下划线 | 记混了 Python 语法 | 改为 `json.loads`（点号） |
| `json.dump(data)` 少参数 | dump 需要两个参数 | `json.dump(data, f)` |
| `KeyError: '分数'` | CSV 列名有空格或中文逗号 | 打印列名确认，CSV 全部用英文逗号 |
| 中文变乱码 | 没加 `ensure_ascii=False` | 写文件时加上 |
| `dict(json.load(f))` 多余 | load 已返回字典 | 直接 `json.load(f)` 即可 |

## 课外知识：json.dumps() 的 default 参数

当 json 不认识自定义对象时，用 `default` 指定转换函数：

```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

def student_to_dict(s):
    return {"name": s.name, "age": s.age}

s = Student("张三", 20)
json.dumps(s, default=student_to_dict)  # 自动把 Student → 字典 → JSON
```
