# Day 30 学习总结：BGE 中文 Embedding 评测

## 今天完成了什么

今天安装了 `sentence-transformers`，并使用 `BAAI/bge-small-zh-v1.5` 做了中文 embedding 评测。

完成内容：

- 验证 `torch`、`sentence-transformers`、`transformers` 能正常导入。
- 加载 BGE 中文 embedding 模型。
- 生成 512 维中文向量。
- 用同一份 FAQ 和同一组 10 题做检索评测。

## 三组结果对比

```text
教学版关键词 embedding：70%
Chroma 默认 embedding：40%
BGE 中文 embedding：70%
```

## 今天的关键理解

中文 embedding 模型并不是一接入就一定超过关键词 baseline。

原因是检索效果不只取决于 embedding，还取决于：

- 文档怎么切分。
- 问题怎么设计。
- expected chunk 是否合理。
- 相近主题是否混在一起。
- 是否需要 rerank 或更细粒度 chunk。

## 当前判断

BGE 中文 embedding 已经证明工程可行，而且比 Chroma 默认 embedding 更适合当前中文 FAQ。

但它还没有超过教学版关键词 baseline，所以暂时不应该直接替换主应用。
