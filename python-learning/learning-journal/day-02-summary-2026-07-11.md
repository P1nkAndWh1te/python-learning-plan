## 今日学习内容

### 函数进阶
- **位置参数**：按顺序传，必须传
- **默认参数**：`def f(a, b=0)`，不传就有默认值
- **可变参数 `*args`**：收"不带等号"的多余参数，打包成元组
- **关键字参数 `**kwargs`**：收"带等号"的多余参数，打包成字典
- **参数顺序**（两种写法都合法）：
  - 常规：`def f(a, b="默认", *args)` → 默认参数在 `*args` 前
  - 进阶：`def f(a, *args, b="默认")` → 默认参数在 `*args` 后，强制用关键字传参

### 模块导入
- `import xxx` → 用的时候带前缀 `xxx.func()`
- `from xxx import yyy` → 直接调用 `func()`
- `import xxx as alias` → 别名 `alias.func()`
- `if __name__ == "__main__":` → 防止被 import 时执行测试代码

### 列表操作
- 增：`append()`（末尾）、`insert(index, val)`（中间）
- 删：`pop()`（末尾）、`pop(index)`（指定位置）、`remove(val)`（指定值）
- 改：`L[0] = xxx`
- 查：`len()`、`in`、`index()`
- 切片：`L[start:end:step]`、`L[::-1]`（倒序）
- 排序：`sort()`、`sort(reverse=True)`、`reverse()`

### 变量作用域
- 函数里**读**全局变量：直接读
- 函数里**改**全局变量：必须声明 `global x`
- 没有 `global` 时，函数里的赋值创建的是局部变量

### PyCharm 技巧
- 选中代码 → 右键 → Execute Selection in Python Console（只跑选中部分）
- `Ctrl + /` 快速注释/取消注释
- 建议每天新建一个 `.py` 文件，避免从头跑



## 踩过的坑

| 坑 | 原因 | 解法 |
|----|------|------|
| import 时执行了 Day1.py 的全部代码 | Python 导入时会执行整个文件 | 测试代码放进 `if __name__ == "__main__":` |
| `print(函数())` 输出 None | 函数没有 return，外面又包了 print | print 和 return 二选一 |
| `(4,)` 后面多个逗号 | 单元素元组必须带逗号，`(4)` 是数字 | 正常现象，不是 bug |
| `len(*args)` 报错 | `*args` 拆开后 len 不接受多个参数 | 写 `len(args)` |
| `*args` 和默认参数互相抢位置 | 位置参数会先填满默认参数 | 默认参数放 `*args` 后面，强制关键字传参 |
