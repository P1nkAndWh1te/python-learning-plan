# Day 28 学习总结：真实 Embedding 最小验证

## 今天完成了什么

今天用独立脚本验证了 Chroma 默认 embedding function。

脚本完成了：

- 读取 `python_learning_faq.md`。
- 复用主应用的 Markdown 标题切分逻辑。
- 使用 Chroma 默认 embedding function 建库。
- 对同一组 10 个问题进行检索评测。
- 将结果和教学版关键词 embedding 的 70% baseline 对比。

## 关键结果

```text
教学版关键词 embedding：70%
Chroma 默认 embedding：40%
```

## 我的理解

真实 embedding 能把文本转换成高维向量，但效果取决于模型是否适合当前语料。

Chroma 默认 embedding 能跑通工程链路，但在当前中文 FAQ 上表现不如教学版关键词 embedding，所以不能直接替换主应用。

## 今天的结论

当前项目应该保留教学版关键词 embedding 作为 baseline，并把 Chroma 默认 embedding 作为对照实验。

下一步应该评估更适合中文的 embedding 模型，而不是只因为它是“真实 embedding”就直接上线。
