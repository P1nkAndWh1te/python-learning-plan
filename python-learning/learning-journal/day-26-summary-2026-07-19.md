## 当前评测结论

当前教学版 embedding 的检索结果：

```text
总题数：10
命中数：7
未命中数：3
命中率：70%
```

未命中主要来自：

- 宽泛问题信息密度不足。
- DeepSeek / API Key / 环境变量等相近主题互相干扰。
- 教学版关键词向量无法理解语义意图。

---

## 今日理解

评测的价值不是证明系统“永远正确”，而是暴露系统当前边界。

如果为了追求 100% 命中而继续堆人工关键词，评测会失真。保留 70% 的真实命中率，反而能为后续真实 embedding 对比提供基线。
