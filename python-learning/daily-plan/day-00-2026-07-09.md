## 二、方向与岗位定位

当前主线选择：**RAG / LLM 数据工程项目**。

这条路线更适合 30 天冲刺，因为它具备三个优势：

- 技术栈集中：Python + Git + SQL + LlamaIndex + Chroma + Streamlit。
- 成果可展示：可以做出一个网页问答系统，而不是只有终端日志。
- 面试可讲：能围绕文档解析、分块、向量化、检索、生成、引用溯源、效果评估展开。

目标岗位不再泛泛写“数据开发 / 数据挖掘 / 数据仓库”三条线同时冲刺，而是收敛为：

| 优先级 | 岗位方向 | 当前策略 |
|---|---|---|
| P0 | AI 应用开发 / LLM 应用实习 | 主投方向，用 RAG 项目证明动手能力 |
| P1 | 数据开发实习 | 补 SQL、Python、Git，项目里强调数据处理链路 |
| P2 | 数据分析 / 数据工程助理 | 用 Pandas、SQL、项目 README 和演示视频增强可见度 |

不建议 30 天内同时深入 Java、Scala、Flink、Kafka、Hive、Docker、算法模型训练。它们都是长期方向，不适合作为这 30 天的主线。
