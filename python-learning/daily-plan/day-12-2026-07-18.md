## 关键理解

LlamaIndex 把 RAG 里的多个步骤封装起来：

```text
文档加载 -> 文档切分 -> embedding -> 索引 -> 检索 -> LLM 生成回答
```

今天用 mock 模型跑通的是“框架链路”。明天再考虑接入真实 LLM / embedding。
