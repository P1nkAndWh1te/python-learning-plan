## 核心概念

### 1. 表、行、列

可以把数据库表理解成 Excel：

```text
students 表
id | name | city | score | course_id
```

- 表：一张表格。
- 行：一条记录。
- 列：一个字段。

### 2. select

查询数据：

```sql
select name, score from students;
```

### 3. where

过滤数据：

```sql
select name, score
from students
where score >= 90;
```

### 4. order by

排序：

```sql
select name, score
from students
order by score desc;
```

### 5. group by

分组统计：

```sql
select city, avg(score)
from students
group by city;
```

### 6. join

连接两张表：

```sql
select students.name, courses.course_name
from students
join courses on students.course_id = courses.id;
```
