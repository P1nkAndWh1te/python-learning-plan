## 今日学习任务

1. **Python 环境搭建**：安装 Python 3.10+，配置 VS Code 或 PyCharm，运行第一行 `print("Hello World")`
2. **变量与数据类型**：掌握 `int`、`float`、`str`、`bool` 四种基本类型，理解 `type()` 和类型转换
3. **条件判断与循环**：掌握 `if/elif/else`、`for` 循环、`while` 循环，理解 `range()` 用法
4. **函数基础**：学会用 `def` 定义函数，理解参数和 `return` 返回值

---

## 今日检验标准

学完之后，你应该能独立完成：
- 写一个函数，接收用户名和年龄，返回"XX 今年 YY 岁"
- 用 `for` 循环打印 1-100 中所有能被 3 整除的数
- 用 `if` 判断一个年份是否是闰年
