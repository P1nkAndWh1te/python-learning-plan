## 今日关键理解

今天的 RAG 链路是：

```text
用户问题
  -> LlamaIndex 检索相关文档片段
  -> 把片段作为 context
  -> DeepSeek 根据 context 生成回答
```

注意：

- LlamaIndex 负责检索。
- DeepSeek 负责生成回答。
- embedding 暂时用本地关键词 embedding。
- 后续需要替换成真实 embedding 模型。
