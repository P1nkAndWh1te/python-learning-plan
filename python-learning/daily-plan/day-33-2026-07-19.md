# Day 33：评测指标展示与 BGE 体验优化

## 今日目标

优化 Streamlit 主应用中的检索评测展示，让页面同时显示：

```text
Top-1 hit
Top-k recall
```

并补充 BGE 模式首次加载较慢的提示。

## 背景

Day32 已经把主应用改成双 embedding 模式：

```text
Teaching keyword embedding
BGE Chinese embedding
```

但页面只显示一个 `Hit rate`，这容易让人误以为检索质量只有一个指标。

## 今日任务

1. 新增 `calculate_top_k_hit_rate`。
2. 页面用两个 metric 展示 top-1 和 top-k。
3. BGE 模式下在侧边栏显示首次加载提示。
4. 新增独立指标检查脚本。
5. 更新 README、计划、复盘和学习总结。

## 今日边界

- 不修改 DeepSeek 生成逻辑。
- 不改变 FAQ 和 10 题评测集。
