# Day 32：主应用接入 BGE 双模式

## 今日目标

把 Day31 验证过的 BGE 中文 embedding 接入 Streamlit 主应用，同时保留教学版关键词 embedding。

## 背景

Day31 修正评测口径后得到：

```text
教学版关键词 embedding：top-1 60%，top-k 100%
Chroma 默认 embedding：top-1 50%，top-k 70%
BGE 中文 embedding：top-1 80%，top-k 100%
```

BGE 是当前效果最好的真实中文 embedding 方案，但教学版关键词 embedding 仍然有教学价值。
