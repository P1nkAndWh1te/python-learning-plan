# Day 29: Chinese Embedding Dependency Check

This folder records the dependency check before trying a Chinese embedding model.

No package is installed by this script.

## Run

From the project root:

```powershell
python "03_执行过程/day29_embedding_dependency_check/dependency_check.py"
```

## Optional dry run

This command resolves dependencies without installing them:

```powershell
python -m pip install --dry-run sentence-transformers
```
