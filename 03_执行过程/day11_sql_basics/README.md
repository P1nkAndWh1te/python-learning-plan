## 运行

```powershell
python "03_执行过程/day11_sql_basics/sql_basics_demo.py"
```

## 今日重点

你需要能解释：

- `select`：选择要看的列。
- `where`：过滤符合条件的行。
- `order by`：排序。
- `group by`：分组统计。
- `join`：把两张表按关系连起来。
